Olympus Method — Training Split

DAY 1 — PUSH (Chest / Shoulders / Triceps)

⚡ First exercise gets warm-ups (30%×5, 50%×5)

  1. Incline Barbell Bench Press
    Warm-ups: ✔
    Working Sets: 6–10 + failure
  2. Flat Barbell Bench Press
    Working Sets: 6–10 Failure
  3. Weighted Dips (Chest-focused)
    Working Sets: 6–10 Failure
  4. Dumbbell Lateral Raises
    Working Sets: 10–15 Failure
  5. Optional: Rope Pushdown or Skull Crushers
    Working Sets: 8–12 Failure

DAY 2 — LEGS (Quads / Hams / Calves)

⚡ Warm-ups only for back squat

  1. Barbell Back Squat
    Warm-ups: ✔
    Working Sets: 6–10 Failure
  2. Romanian Deadlift
    Working Sets: 6–10 Failure
  3. Hamstring Curl (Machine)
    Working Sets: 8–12 Failure
  4. Calf Raises
    Working Sets: 10–15 Failure
  5. Optional: Leg Press or Hack Squat
    Working Sets: 8–12 Failure

DAY 3 — PULL (Back / Rear Delts / Biceps)

⚡ Warm-ups only for Weighted Pull-Ups

  1. Weighted Pull-Ups
    BW × 5
    BW × 5
    Working Sets: 6–10 Failure
  2. Bent-Over Barbell Row
    Working Sets: 6–10 Failure
  3. Barbell Biceps Curl
    Working Sets: 6–10 Failure
  4. Rear Delt Flys
    Working Sets: 10–15 Failure
  5. Optional: Barbell Shrugs
    Working Sets: 8–12 Failure
